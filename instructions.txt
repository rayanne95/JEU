Pour s'échapper de la maison de Granny il faut récupérer le code secret via des commandes linux !
Les instructions sont les suivantes : 

Copier le fichier texte.txt dans un nouveau fichier appelé texte1.txt

Ecrire la commande 'chmod u+x <fichier.sh> ' pour tout les fichier sh

Lancer start.sh

Modifier texte1.txt 
  retirer la premiere ligne et le premier paragraphe

Compter le nombre de mot du fichier (parmi ce nombre il y a le premier chiffre du code secret)

Pour trouver le bon chiffre s'aider du nombre affichié dans le texte (ils ne sont pas les bons...)

De plus, le chiffre représentant le nombre de fichier txt initial n'est également pas le bon.

Si tu modifies l'histoire ou les codes, tout sera perdu. Bonne chance.
